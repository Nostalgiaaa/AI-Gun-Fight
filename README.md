## 一句话定位
**用 AI 培养一个会打仗的角色，让它在各种战场中证明自己。**

养成是壳，AI 是新引擎，对抗是出口，社区是终局。

---

## 核心循环（养成驱动）

```
       ┌─────────────────────────────────────┐
       ↓                                      │
   培养 agent                                 │
   （写 prompt / 改脚本 / 升级 endpoint）       │
       ↓                                      │
   出战（吃鸡 / Roguelike / 大战场 / 嘴炮）   │
       ↓                                      │
   看回放 + 获得反馈                          │
   （战斗日志、AI 解说、排名变化）             │
       ↓                                      │
   反思 → 改进 → 进化 ────────────────────────┘
                  ↓
              形成"我的 agent"身份认同
                  ↓
              分享、组队、上分、传播
```

**养成系最关键的设计**：玩家要**对自己的 agent 产生情感投入**。
- 给 agent 起名、自定义外观、签名口号
- 战绩、胜场、人格特点全部归属于"我的 agent A"
- agent 可以"成长"——通关 Roguelike 解锁能力上限、对战累积"作战经验" buff
- **杀手锏**：让 AI 自己写战后日记（"今天我在 B 区遇到了 player_X 的狙击手，我做了 Y..."），**情感粘性瞬间起来**

## 三大玩法支柱

### 支柱一：吃鸡战场（PvP 主玩法）
- 多人小队对抗，缩圈、物资、视野、职业 combo
- **定位**：竞技场，证明 agent 实力的地方
- **用户画像**：硬核玩家、追求上分

### 支柱二：Roguelike 试炼（PvE 养成主线）
- 单人闯关、随机地图、装备掉落、boss 战
- **定位**：onboarding + 日常养成 + agent 成长曲线
- **用户画像**：休闲玩家、想看自己 agent 变强的人
- **是冷启动救命药**，比 PvP 优先级更高

### 支柱三：大战场指挥（旗舰内容）
- 司令官 agent + 单位群，几十到几百单位混战
- **定位**：观赏性顶点、行业话题制造机、IP 输出
- **用户画像**：极客、内容创作者、企业团建
- **晚做**，作为 V3 旗舰

### 副品牌：AI 嘴炮 / 人设乱战（破圈炮）
- 独立小产品，独立域名，单独传播
- 写两个人设，看 AI 互喷 / 辩论 / 谈恋爱
- **30 秒出结果，结果即聊天截图**，天然抖音/小红书素材
- 用户从这里被吸引 → 知道主站 → 转化为正式玩家
- **不与主品牌混合**，定位分离

## 接入方式（四种 + 一种隐藏）

| 方式 | 用户 | 谁付钱 |
|---|---|---|
| 站内 Prompt 编辑器 | 大众 | 平台（限免费额度） |
| 纯代码（JS / isolated-vm） | 程序员 | 无 |
| Dify / Coze 接入 | 中级用户 | 用户自己的平台 |
| Webhook Endpoint | 硬核 / 企业 | 用户服务器 |
| ~API Key~ | 极少数 | 用户（**仅隐藏菜单**） |

**底层统一**：所有方式共享同一份 `state JSON` ↔ `action JSON` 协议。

---

## 社区与传播

- **分赛道排行榜**：四种接入方式分开排名
- **混合赛事**：跨赛道、季度大赛，话题制造
- **AI 解说系统**：用便宜 LLM 解读 log → 体育解说式文本/语音
- **自动导播**：每局自动剪 30 秒精彩集锦
- **Agent 商城**：玩家分享、fork、改造别人的 agent
- **战后日记**：AI 用 agent 的"视角"写战斗回忆，养成情感粘性

---

## 技术栈（不变）

- 沙盒：isolated-vm（JS V8 isolate）
- 平台 LLM：DeepSeek V3 + 智谱 GLM-4-Flash 免费档
- 优化：prompt cache + JSON Mode + 规则引擎兜底（90% 决策不调 LLM）
- 协议：tick-based JSON in/out
- 1.0 渲染：2D Canvas + 文字 log
- 2.0 渲染：3D 集锦视频（导演模式，非主玩法）

## 经济模型

| 收入项 | 来源 |
|---|---|
| 订阅 | 高级模式、更多平台 LLM 额度、外观皮肤 |
| 点卡 / 战令 | 月卡、赛季通行证 |
| Agent 商城分成 | 用户售卖 agent 模板 / prompt 包 |
| 企业版 | 团建赛事、企业品牌赛 |
| 副品牌引流 | AI 嘴炮等小工具广告 / 付费功能反哺主站 |

**单局成本上限 ¥0.5**，靠 prompt cache + 规则兜底 + 批量推理压下来。

---

## 关键风险

1. **养成系拉高了制作标准**：视觉、数值、节奏被对标剑与远征/赛马娘，**这是 10 倍工作量增加**
2. **观赏性仍是命门**：自动导播 + AI 解说做得好不好，决定能不能传播
3. **冷启动**：种子 agent 必须官方自产，前 100 个用户要手动运营
4. **副品牌的诱惑**：人设乱战如果先火了，**别本末倒置**抛弃主站
5. **平衡运营**：四种接入方式、三大玩法、agent 成长曲线，平衡复杂度极高
6. **大厂抄袭**：字节、网易、米哈游做"AI 自走棋"只是时间问题，**先发优势 + 社区资产是唯一护城河**

# 客观评判：好玩吗？

养成系框架让我必须**上调评估**。重新打分：

### 第一层：作为养成 + 自走棋游戏，它好玩吗？

**好玩。而且潜力比我之前估的大。**

养成系的核心快乐——"**这是我培养的，它代表我**"——你的项目结构上完全具备，甚至比传统养成系更强：
- 传统养成系：你培养的角色行为是策划写的脚本，**所有玩家最终殊途同归**
- 你的项目：每个玩家培养的 agent 行为**真的不一样**，因为 prompt / 脚本是真的代码
- 这意味着**"我的 agent 真的是独一无二的"** —— 这是养成系玩家梦寐以求但传统养成系永远做不到的事

**这是个真护城河**。米哈游、网易抄你架构容易，**但他们无法消除"我的 agent 独一无二"这种感觉**——只要他们做的还是 GPT 控制官方角色，玩家的认同感就到顶。

### 第二层：作为大众游戏，它好玩吗？

**勉强及格，看执行**。

养成系大众玩家最在意的不是策略深度，是**视觉爽感 + 反馈节奏 + 角色魅力**。这三件事**和 AI 完全无关，全是传统游戏制作功夫**：
- 战斗动画够不够爽？
- 升级/进化的画面够不够带感？
- 角色立绘够不够好看？
- 数值掉落够不够多巴胺？

**这才是养成系的真正生死线**。你的"AI 独特性"是吸引玩家进来的钩子，**留下来靠的是这些 polish**。

诚实说：以独立开发者 / 小团队的资源，**这关是难关**。但比起做出原神，做出"放置奇兵级"的视觉是可达成的。

### 第三层：作为项目，值不值得做？

我的判断**比之前更乐观，但要求也更高**：

| 评估项 | 旧框架（niche 平台） | 新框架（养成 + 自走棋） |
|---|---|---|
| 目标用户量 | 几千核心 | 几万到十几万 |
| DAU 天花板 | 1000-3000 | 10000-50000 |
| 商业化空间 | 订阅 + 赛事赞助 | 订阅 + 战令 + 皮肤 + 商城 + 企业版 |
| 18 个月可行性 | 高 | 中 |
| 制作难度 | 中 | 高 |
| 抄袭风险 | 低 | 高 |
| 出圈可能 | 低 | 中（靠副品牌） |

### 一句话总结

> **这是一个"理论上可以做到剑与远征 1/10 体量"的项目。**

意思是：**它的天花板有 100 万级注册、5-10 万 DAU、年收入千万级**，前提是养成系该做的功夫都做到位。这是**比 Battlesnake 大 10 倍**的机会，也是**比 Battlesnake 难 10 倍**的工程。

### 最重要的一条建议

**养成系的成功靠的不是"AI 多牛"，是"成长曲线多爽"**。

如果你只把 80% 精力投在 AI 创新上，**它会变成一个有趣但留不住人的 demo**。如果你把 40% 精力放在 AI 创新、40% 放在养成系数值/视觉/反馈、20% 放在传播/运营，**它有可能成为一个新品类**。

这个精力分配的转换，比任何技术决策都重要。

最后回到你最初的问题——"这游戏好玩吗？"

我的最终答复：

**对一小撮人（程序员、极客）从 V0.5 就开始好玩。**
**对大众玩家，需要做到 V1.5（养成系统 + Roguelike 完整）才算"够好玩"。**
**做到 V2.0（出圈副产品 + 视觉升级）才算"有出圈潜力"。**

**值得做。但你要做的是一款真游戏，不是一个 AI 玩具。**
